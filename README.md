# BLEduino Android App

The official Android app for the BLEduino development board.  Features 4 modules to quickly interact with the device.

1. Keyboard - Use your phone to emulate a computer keyboard.
2. Console - A chat application between the computer and the phone.
3. LED - Turn on/off every GPIO of the BLEduino.
4. Notification - Send a message from the computer to the phone, and it will be displayed as a notification.
